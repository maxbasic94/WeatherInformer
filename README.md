## How to use

When you start the application, it will show you the weather in the city in which you are now\
On home page you can see:
* Current city
* Temperature
* Condition
* Condition's description
* Humidity
* Pressure
* Wind speed
* Sunrise and sunset times
* Today and next 2 days forecast
